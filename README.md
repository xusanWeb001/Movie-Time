# 🎬 Movie Time

A clean movie listing app built with vanilla HTML, CSS, and JavaScript.

## 🔗 Live Demo
[movie-time-sigma-red.vercel.app](https://movie-time-sigma-red.vercel.app)

## 🛠️ Tech Stack
- HTML5
- CSS3 (Grid, Animations)
- Vanilla JavaScript
- TMDB Movie Data

## ✨ Features
- 44 movies displayed in a responsive grid
- Search by movie title (real-time)
- Filter by genre
- Skeleton loading animation
- Smooth hover effects
